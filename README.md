# ExcelWeb

A simple Flask application for viewing Excel spreadsheets in the browser.

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Place your `.xlsx` files in the `spreadsheets/` directory.
3. Run the development server:
   ```bash
   python -m excelweb
   ```
4. Open your browser at [http://localhost:5000](http://localhost:5000).
